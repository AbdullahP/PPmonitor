How to update {{ shop_id }}.com cookies

  1. Open Chrome and go to bol.com — browse any product page
  2. Open DevTools (F12) → Application tab → Cookies → bol.com
  3. Find these cookies: _abck, ak_bmsc, bm_sz, RT
  4. Or use this snippet in DevTools Console:
JSON.stringify(document.cookie.split(';').map(c => {
  const [name, ...rest] = c.trim().split('=');
  return {name: name.trim(), value: rest.join('=') || '', domain: '.bol.com'};
}))

Or install the EditThisCookie or Cookie-Editor extension and export as JSON.
Cookies typically last 2-7 days before needing a refresh.
